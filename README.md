# Ristinolla
Kyseessä on Windows Forms App-ohjelma, jolla kaksi pelaajaa voi pelata klassista ristinollaa samalta laitteelta. Esittelen nyt hieman ohjelmaa.

## Selvitys sovelluksen toiminnasta
